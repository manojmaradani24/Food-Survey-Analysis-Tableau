# 🍽️ Food Survey Analysis using Tableau

## 📌 Project Title

**Comprehensive Analysis and Dietary Strategies with Tableau: A College Food Choices Case Study**

## 📖 Project Overview

This project was developed as part of the **Skill Wallet Group Project**. It analyzes the dietary habits, food preferences, and lifestyle choices of college students using Tableau. The dataset contains survey responses from students and is transformed into interactive dashboards and stories to identify trends and provide meaningful insights into nutrition and student wellness.

## 🎯 Objectives

* Analyze college students' food habits.
* Create meaningful visualizations using Tableau.
* Build an interactive dashboard and story.
* Present insights to support healthy dietary awareness.

## 🛠️ Technologies Used

* Tableau Public
* Git & GitHub
* CSV Dataset
* Data Visualization

## 📂 Dataset

* **food_coded.csv** – Main dataset
* **codebook_food.docx** – Dataset documentation (codebook)

## 📊 Visualizations

* Breakfast Analysis
* Gender Distribution
* GPA Distribution
* Weight vs GPA
* GPA by Gender

## 📈 Dashboard

An interactive Tableau dashboard presenting key insights from the College Food Choices dataset.

## 📖 Story

A Tableau Story was created to explain the visualizations and findings step by step.
